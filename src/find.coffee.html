Find files

This method will list all files and directories in the given directory.

Arguments:

  • source Path to be searched.
  • options Specification of files to find.
  • callback(err, list) The callback will be called just if an error occurred. The list of found entries will be given.
  • depth Search depth as integer (internal parameter).

Find files (Synchronous)

This method will list all files and directories in the given directory.

Arguments:

  • source Path to be searched.
  • options Specification of files to find.
  • depth Search depth as integer (internal parameter).

Return:

  • list Returns the list of found entries

Throw:

  • Error If anything out of order happened.
 
findSync = module.exports.sync = (source, options = {}, depth = 0) ->
